# Yulin Luo's Personal Homepage

Academic personal homepage of [Yulin Luo](https://yulin-luo.github.io/), based on the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) Jekyll theme.

## Live Site

- **URL**: https://yulin-luo.github.io/

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://127.0.0.1:4000.
